Connect Google Drive

Sign in with Google and sync your tracker data to Drive — accessible from any device, automatically backed up. You need a free Google OAuth Client ID (takes ~3 minutes).

1
Open console.cloud.google.com → create a project → go to APIs & Services → Credentials
2
Click Create Credentials → OAuth 2.0 Client ID → Web application
3
Under Authorised JavaScript origins add your site URL (e.g. https://defaulte-r.github.io)
4
In APIs & Services → Library enable the Google Drive API
5
Copy your Client ID below and click Save & Sign In
Already have a Client ID saved? Just click Sign In above — it will use the saved one.
