Connect Google Drive

Sign in with Google and sync your tracker data to Drive — accessible from any device, automatically backed up.

1
Go to Google Cloud Console → create a project → APIs & Services → Credentials
2
Click Create Credentials → OAuth 2.0 Client ID → Web application
3
Add your site URL to Authorised JavaScript origins (e.g., https://yourusername.github.io)
4
Enable the Google Drive API in APIs & Services → Library
5
Copy your Client ID and paste it below
Already have a Client ID saved? Just click Save & Sign In — it will use the saved one.
